Testing an Ad Frame

With only required attributes. Frame name appears as title. No sub-title.

Show powered by.

With data-ad-title and data-ad-sub-title. Frame name is overridden.

With data-hide-title enabled. No frame title appears.

With data-product-count set to 2. Only 2 products appear.

With data-product-count set to 10. 4 products appear (frame has 4 products).

With data-click-target set to '_self' instead of the default of '_blank'. Clicking on a product opens destination frame in same tab.

With data-tracking set to "abc=123&xyz=789". Clicking on a product will open destination frame and parsed parameters will be vibile under page description.
