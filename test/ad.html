Testing an Ad Frame

With only required attributes. Frame name appears as title. No sub-title.

With only data-ad-title and data-ad-sub-title. Frame name is overridden.

With only data-hide-title enabled. No frame title appears.

With only data-product-count set to 2. Only 2 products appear.

With only data-product-count set to 4 and data-hide-out-of-stock set to 0. 4 products appear.

With only data-product-count set to 4 and data-hide-out-of-stock set to 1. 3 products appear.

With data-click-target set to '_self' instead of the default of '_blank'. Clicking on a product opens destination frame in same tab.

With data-tracking set to "abc=123&xyz=789". Clicking on a product will open destination frame and parsed parameters will be vibile under page description.
